Мульти-кабинетная сборка WB BOT

Что добавлено:
- 5+ кабинетов в одной панели
- вкладки кабинетов наверху
- страница /tenants для добавления, обновления, входа и сброса кабинетов
- отдельные cookies/storage_state, очереди, черновики, архивы и бэкапы на каждый кабинет
- complaint_worker.py умеет работать по одному кабинету или по всем последовательно
- login_wb.py умеет сохранять авторизацию отдельно для выбранного кабинета

Как начать:
1. Запустите python app.py
2. Откройте /tenants
3. Добавьте кабинет
4. Нажмите «Войти в WB» для кабинета
5. После авторизации откройте вкладку нужного кабинета и работайте как раньше

Данные кабинетов лежат в PRIVATE_ROOT/tenants/<tenant_id>/
