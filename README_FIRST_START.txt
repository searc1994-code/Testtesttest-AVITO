Быстрый старт универсальной сборки
=================================

1. Установите зависимости:
   pip install -r requirements.txt
   playwright install chromium

2. Запустите приложение:
   python main.py

3. При первом запуске откроется страница /setup-admin.
   На ней нужно один раз задать пароль администратора.
   Логин по умолчанию: admin

4. Пароль хранится не в открытом виде, а в виде хеша.
   Файл инициализации лежит в приватной папке:
   WB_PRIVATE_DIR/security/admin_auth.json

5. Если нужно, можно задать пароль через переменную окружения:
   APP_ADMIN_PASSWORD=...
   Тогда страница первичной настройки не понадобится.

6. Прогон базовых проверок:
   python checks/run_all.py

7. Проверка зависимостей:
   Скрипт checks/check_dependencies.py сравнивает установленное окружение
   с версиями из requirements.txt.
